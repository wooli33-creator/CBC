# 기후 위기 빙고 챌린지 🌏

성인 학습자를 위한 교육용 빙고 게임으로, 기후 변화 지식을 재미있게 학습할 수 있습니다.

## 게임 모드

### 1. 모둠 모드 (여러 사람) 👥

**특징:**
- 각 사용자마다 고유한 빙고판이 생성됩니다 (시드 기반 랜덤)
- 5단계 레벨 시스템 (3×3 → 4×4 → 5×5 → 6×6 → 7×7)
- 각 레벨에서 **3줄 이상** 완성 시 다음 단계로 자동 진행
- 최종 목표: "지구 지킴이 등단" 달성

**게임 방법:**
1. 3×3 그리드에서 시작 (연습 단계)
2. 타일을 클릭하여 기후 키워드 학습
3. 3줄(행/열/대각선)을 완성하면 레벨 완료
4. 5단계를 모두 완료하면 최종 인증

**레벨 구성:**
- **레벨 1** (3×3): 연습 단계 - 도담도담 정원, 기후미식 등
- **레벨 2** (4×4): 초급 지킴이
- **레벨 3** (5×5): 중급 지킴이
- **레벨 4** (6×6): 상급 지킴이
- **레벨 5** (7×7): 마스터 지킴이

### 2. 혼자하기 (컴퓨터 대전) 🤖

**특징:**
- 플레이어 vs 컴퓨터 대결
- 그리드 크기 선택 가능 (3×3 ~ 7×7)
- 턴 기반 "단어 뽑기(Draw)" 시스템
- 먼저 **3줄**을 완성하는 쪽이 승리

**게임 방법:**
1. 원하는 그리드 크기 선택 (3×3 ~ 7×7)
2. "단어 뽑기" 버튼 클릭
3. 뽑힌 단어가 플레이어/컴퓨터 보드에 있으면 자동으로 체크됨
4. 먼저 3줄을 완성한 쪽이 승리
5. 동시에 완성하면 무승부

**승리 조건:**
- 플레이어가 먼저 3줄 완성 → **플레이어 승리** 🎉
- 컴퓨터가 먼저 3줄 완성 → **컴퓨터 승리** 💻
- 동시에 3줄 완성 → **무승부** 🤝

## 주요 기능

### 교육적 콘텐츠
- **70+ 기후 관련 키워드**: 탄소중립, 재활용, 태양에너지, 지구온난화 등
- **상세한 설명**: 각 키워드를 클릭하면 자세한 설명 제공
- **난이도 조절**: 3×3 연습 모드부터 7×7 전문가 모드까지

### 게임 시스템
- **시드 기반 랜덤화**: 각 세션마다 고유한 보드 생성
- **진행도 추적**: 실시간으로 완성된 줄 수 표시
- **반응형 디자인**: 데스크톱/모바일 모두 지원

### 기술적 특징
- 빙고 판정 로직: 행/열/대각선 모두 검사
- 중복 없는 줄 카운팅: `getCompletedLines()` 함수
- 시드 기반 셔플: `seededShuffle()` 함수
- 쉬운 난이도 조절: `REQUIRED_LINES = 3` 상수

## 기술 스택

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **UI Components**: Shadcn UI + Radix UI
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Backend**: Express.js (정적 파일 서빙)

## 시작하기

```bash
# 의존성 설치

# 개발 서버 실행
npm run dev
```

## 코드 구조

### 핵심 함수

```typescript
// 보드 초기화
initBoard(size: GridSize, seed: string, wordArray: KeywordData[]): KeywordData[]

// 완성된 줄 계산
getCompletedLines(selected: Set<number>, size: GridSize): number[][]

// 보드에서 단어 마킹 (혼자하기 모드)
markIfExists(board: KeywordData[], word: string, selected: Set<number>): number[]

// 시드 기반 셔플
seededShuffle<T>(array: T[], seed: string): T[]
```

### 설정 변경

```typescript
// 승리 조건 변경 (기본값: 3줄)
const REQUIRED_LINES = 3;  // 이 값을 변경하면 난이도 조절 가능
```

## 단어 목록 관리

### 연습용 키워드 (3×3)
- 위치: `PRACTICE_KEYWORDS` 배열
- 예시: 도담도담 정원, 기후미식, 걷기운동 등

### 기후 키워드 (4×4 ~ 7×7)
- 위치: `CLIMATE_KEYWORDS` 배열
- 70+ 개의 전문 기후 용어
- 예시: 탄소중립, 재생에너지, 생물다양성 등

## 접근성

- 모든 버튼에 `aria-label` 적용
- 키보드 포커스 스타일 지원
- 충분한 색상 대비
- `data-testid` 속성으로 테스트 용이성 확보

## 사이즈별 동작 점검 항목

### 자체 테스트 체크리스트

개발 또는 수정 후 다음 항목을 확인하세요:

#### 그리드 생성 검증
- [ ] **3×3**: 칸 9개 정확히 생성, 단어 9개 배치 (PRACTICE_KEYWORDS 사용)
- [ ] **4×4**: 칸 16개 정확히 생성, 단어 16개 배치 (CLIMATE_KEYWORDS 사용)
- [ ] **5×5**: 칸 25개 정확히 생성, 단어 25개 배치 (CLIMATE_KEYWORDS 사용)
- [ ] **6×6**: 칸 36개 정확히 생성, 단어 36개 배치 (CLIMATE_KEYWORDS 사용)
- [ ] **7×7**: 칸 49개 정확히 생성, 단어 49개 배치 (CLIMATE_KEYWORDS 사용)

#### 빙고 판정 검증
- [ ] 각 크기에서 가로줄 완성 시 정확히 인식
- [ ] 각 크기에서 세로줄 완성 시 정확히 인식
- [ ] 각 크기에서 대각선 (↘, ↙) 완성 시 정확히 인식
- [ ] 중복된 줄이 여러 번 카운트되지 않음
- [ ] 진행도 UI가 `X / 3` 형식으로 정확히 표시

#### 초기화 검증
- [ ] **Reset 버튼**: 보드/선택상태/진행도 완전 초기화, 새 시드로 보드 재생성
- [ ] **size 변경**: 새 크기로 보드 재생성, 이전 상태 완전 초기화
- [ ] **모드 전환**: 각 모드의 독립적인 상태 유지

#### 단어 부족 경고
- [ ] CLIMATE_KEYWORDS가 `size * size`보다 적으면 경고 메시지 표시
  - 현재 키워드 개수: **51개** (최대 7×7 = 49개 지원 가능)
  - 8×8 이상 선택 시 경고 표시되어야 함

#### 시드 유지
- [ ] 새로고침 시 같은 세션은 동일 배열 표시
- [ ] 새 세션(브라우저 재시작)은 다른 배열 표시
- [ ] Reset 시 새로운 배열 생성

#### 반응형 확인
- [ ] 데스크톱: 큰 그리드에서도 타일 크기 적절
- [ ] 모바일: 폰트/이모지 크기가 넘치지 않음
- [ ] 7×7에서도 타일이 화면에 모두 표시됨
