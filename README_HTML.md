# 기후 위기 빙고 챌린지 - HTML 버전

## 사용 방법

`bingo.html` 파일을 브라우저에서 직접 열어서 사용하세요!

### 방법 1: 파일 직접 열기
1. `bingo.html` 파일을 찾습니다
2. 파일을 더블클릭하거나 브라우저로 드래그합니다
3. 바로 게임이 시작됩니다!

### 방법 2: 로컬 서버 사용
```bash
# 간단한 HTTP 서버 실행
python3 -m http.server 8000

# 브라우저에서 열기
http://localhost:8000/bingo.html
```

## 게임 모드

### 모둠 모드
- 3×3 연습 단계부터 7×7 마스터 단계까지 5단계 진행
- 각 레벨에서 3줄을 완성하면 다음 단계로 진행
- 잘못 선택한 타일은 다시 클릭해서 취소 가능
- 모든 레벨 완료 시 "환경지킴이 인증" 획득!

### 혼자하기 모드
- 원하는 그리드 크기 선택 (3×3 ~ 7×7)
- 컴퓨터와 빙고 대결
- "단어 추첨" 버튼을 눌러 랜덤 단어 선택
- 먼저 3줄을 완성하는 쪽이 승리!

## 특징

✅ **완전한 독립 실행**: 인터넷 연결 없이도 작동 (한글 폰트만 온라인 로드)
✅ **모든 기기 지원**: PC, 태블릿, 스마트폰 모두 지원
✅ **즉시 사용 가능**: 설치나 빌드 과정 불필요
✅ **교육용 최적화**: 기후 위기 관련 50개 키워드와 상세 설명
